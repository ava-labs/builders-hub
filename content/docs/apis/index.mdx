## Getting Started with APIs

### Prerequisites

- A running AvalancheGo node or access to a public RPC endpoint
- Basic understanding of JSON-RPC protocol
- API client or tool like curl, Postman, or your preferred programming language

### API Format

Most Avalanche APIs use the JSON-RPC 2.0 format. Here's a basic example:

```bash
curl -X POST --data '{
    "jsonrpc": "2.0",
    "method": "info.getNodeVersion",
    "params": {},
    "id": 1
}' -H 'content-type:application/json;' https://api.avax.network/ext/info
```

### Common Endpoints

- **C-Chain**: `/ext/bc/C/rpc` - Ethereum-compatible RPC
- **P-Chain**: `/ext/bc/P` - Platform Chain operations  
- **X-Chain**: `/ext/bc/X` - Asset creation and transfers
- **Info**: `/ext/info` - Node and network information
- **Health**: `/ext/health` - Health checks

For detailed examples and method documentation, explore the specific API sections above.
