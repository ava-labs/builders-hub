### Get Testnet AVAX

To obtain free testnet AVAX for development and testing:

1. **[Toolbox Faucet](/tools/l1-toolbox/faucet)** - Request testnet tokens directly with a BuilderHub account
2. **[Core Wallet Faucet](https://faucet.avax.network/)** - Request a faucet coupon if you don't have mainnet AVAX
3. **[Guild](https://guild.xyz/avalanche)** - Join the Guild and request a faucet coupon if you don't have mainnet AVAX

<Callout title="Note">
If you already have AVAX on Mainnet, you can use that address to request testnet tokens from the faucet.
</Callout>

### Add Networks to Wallet

Both Mainnet and Testnet explorers have an "Add to Wallet" button for easy MetaMask/wallet integration:

- **Mainnet**: Visit the [C-Chain explorer](https://subnets.avax.network/c-chain) 
- **Testnet**: Visit the [Fuji C-Chain explorer](https://subnets-test.avax.network/c-chain)

Look for "Add Avalanche C-Chain to Wallet" under the "Chain Info" section.

### Architecture

Avalanche uses a unique [consensus mechanism](/docs/primary-network/avalanche-consensus) and consists of:

- **[Primary Network](/docs/primary-network)**: The main network with X, P, and C-Chains
- **[Avalanche L1s](/docs/avalanche-l1s)**: Custom blockchains with their own rules and tokens
- **[Virtual Machines](/docs/avalanche-l1s/virtual-machines)**: Execution environments for different use cases
