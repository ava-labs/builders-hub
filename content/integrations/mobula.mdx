## Overview

Mobula is a low latency, real-time onchain data provider built for developers building any kind of Web3 application. Through a unified API, Mobula delivers stable and accurate onchain data, wallet data and trading data across multiple chains.

## Features

- **Low latency, real time data**: Built to have real time experiences where data freshness matters.
- **Unified API**: One consistent interface for token, market, pool/pair, trade, and wallet data.
- **Stable and accurate results**: Designed for production use cases that need reliable data at scale.
- **Multi-chain coverage**: Support more than 100 blockchains, including Avalanche C-Chain and Avalanche L1 ecosystems.
- **Built for builders**: Clear docs, guides, and a developer-first API surface.

## Use Cases

- **Real time token data**: Power token pages and analytics with real time prices, liquidity, pools, trades, and more.
- **Portfolio and wallet apps**: Build multi-chain portfolio views with balances, positions, history, and activity.
- **Trading apps and infrastructure**: Support trading experiences with low latency market and pool data that updates in real time.
- **Any Web3 app that needs low latency data**: Ideal for products that require stable, accurate, real time onchain data.

## Conclusion

Mobula provides low latency, real time onchain data through a unified API to power any Web3 app across multiple chains. With a broad set of endpoints, developer guides, and a hands-on Demo UI, Mobula helps teams go from prototyping to production without building and maintaining their own indexing and data infrastructure.
