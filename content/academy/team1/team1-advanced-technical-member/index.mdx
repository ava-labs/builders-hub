## Welcome to the Advanced Technical Team1 Path

The **Technical Team1** course defined a baseline: blockchain and Avalanche fundamentals, Solidity, ERC standards, frontend integration, and custom L1s. This **Advanced Technical Team1** course builds on that foundation with three more pillars that come up constantly in ecosystem workshops, hackathons, and builder support:

- **Interoperability on Avalanche (ICM and ICTT)**: sending messages and tokens between Avalanche L1s. Covered in the Academy tracks on [Interchain Messaging](https://build.avax.network/academy/avalanche-l1/interchain-messaging), [ERC20 Bridge](https://build.avax.network/academy/avalanche-l1/erc20-bridge), and [Native Token Bridge](https://build.avax.network/academy/avalanche-l1/native-token-bridge).
- **Encrypted tokens (eERC)**: privacy-preserving tokens built on zero-knowledge proofs and partial homomorphic encryption. Covered in the [Encrypted ERC](https://build.avax.network/academy/encrypted-erc) Academy course.
- **x402 payment infrastructure**: HTTP-native payments for web services and AI agents on Avalanche. Covered in the [x402 Payment Infrastructure](https://build.avax.network/academy/blockchain/x402-payment-infrastructure) Academy course.

Each section of this course explains the concepts thoroughly so you can answer builder questions, design solutions, and run workshops on these topics with confidence.
