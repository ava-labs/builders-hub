### Query the PoS Validator Set

Before performing any staking operations, let's take a look at the current validator set and any existing delegations.

This tool queries validators from the AvaCloud API and enriches them with on-chain staking details from the Staking Manager contract, including:

- **Stake amount** (converted from weight)
- **Delegation fee** percentage
- **Minimum stake duration** and time remaining
- **Uptime** tracking
- **Your delegations** (if wallet is connected)

Feel free to come back to this step after [registering a validator](/academy/permissionless-l1s/06-staking-manager-operations/04-register-validator-demo), [delegating](/academy/permissionless-l1s/06-staking-manager-operations/06-delegate-to-validator-demo), or [removing a validator](/academy/permissionless-l1s/06-staking-manager-operations/08-remove-validator-demo) to verify the changes have applied.
