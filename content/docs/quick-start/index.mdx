## Faucet

- [Console Faucet](/console/primary-network/faucet) — request testnet AVAX with a BuilderHub account
- [Core Wallet Faucet](https://faucet.avax.network/) — get a faucet coupon if you don't have mainnet AVAX
- [Guild](https://guild.xyz/avalanche) — join and request a faucet coupon if you don't have mainnet AVAX

<Callout title="Note">
If you already have AVAX on Mainnet, you can use that address to request testnet tokens from the faucet.
</Callout>
