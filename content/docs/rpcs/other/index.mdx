# Other RPC APIs

This section contains documentation for additional RPC APIs that support AvalancheGo node operations, monitoring, and administration.

## Available APIs

### Node Management
- **Admin API**: Administrative functions for node management
- **Health API**: Health check and monitoring endpoints
- **Info API**: Node information and network status

### Data & Indexing
- **Index API**: Transaction and block indexing services
- **Metrics API**: Prometheus-compatible metrics endpoints

### VM-Specific
- **ProposerVM API**: ProposerVM-specific RPC methods
- **Subnet-EVM API**: Subnet-EVM blockchain RPC methods

### Additional Resources
- **Standards**: RPC standards and conventions
- **Guides**: Implementation guides and best practices
