## Overview

Avalanche supports different networks (mainnet, testnet, custom) with specific address formats. Each network uses different Human-Readable Prefixes (HRP) for XP addresses.

## Address Formats

### EVM Addresses (C-Chain)

EVM addresses are consistent across all networks:

```typescript
const evmAddress = account.getEVMAddress();
// Example: "0x742d35Cc6634C0532925a3b8D4C9db96C4b4d8b6"
```

### XP Addresses (X-Chain and P-Chain)

XP addresses use network-specific HRPs:

```typescript
// Mainnet addresses
const mainnetXAddress = account.getXPAddress("X", "avax"); // X-avax1...
const mainnetPAddress = account.getXPAddress("P", "avax"); // P-avax1...

// Testnet addresses
const fujiXAddress = account.getXPAddress("X", "fuji"); // X-fuji1...
const fujiPAddress = account.getXPAddress("P", "fuji"); // P-fuji1...
```

## Network Configuration

### Mainnet (Production)

```typescript
import { avalanche } from "@avalanche-sdk/client/chains";
import { privateKeyToAvalancheAccount } from "@avalanche-sdk/client/accounts";
import type { AvalancheAccount } from "@avalanche-sdk/client/accounts";

const account: AvalancheAccount = privateKeyToAvalancheAccount("0x...");

// Mainnet addresses
const mainnetAddresses = {
  evm: account.getEVMAddress(),
  xChain: account.getXPAddress("X", "avax"),
  pChain: account.getXPAddress("P", "avax"),
};

console.log("Mainnet addresses:", mainnetAddresses);
```

import { avalancheFuji } from "@avalanche-sdk/client/chains";

## Network Detection

### Detect Network from Chain

```typescript
import { avalanche, avalancheFuji } from "@avalanche-sdk/client/chains";

function getNetworkHRP(chainId: number): string {
  switch (chainId) {
    case avalanche.id:
      return "avax";
    case avalancheFuji.id:
      return "fuji";
    default:
      return "avax"; // Default to mainnet
  }
}

try {
  const hrp: string = getNetworkHRP(client.chain.id);
  const xChainAddress: string = account.getXPAddress("X", hrp);
  console.log("X-Chain Address:", xChainAddress);
} catch (error) {
  console.error("Failed to get network HRP:", error);
}
```
