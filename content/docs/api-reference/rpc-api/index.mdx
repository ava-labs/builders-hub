# AvalancheGo RPC APIs

AvalancheGo nodes expose a comprehensive set of RPC APIs for interacting with the Avalanche network. These APIs allow you to:

- **Platform Chain (P-Chain)**: Manage validators, subnets, and staking
- **Exchange Chain (X-Chain)**: Create and transfer assets
- **Contract Chain (C-Chain)**: Deploy and interact with smart contracts
- **Admin & Node Management**: Control and monitor node operations
- **Health & Metrics**: Check node status and performance
