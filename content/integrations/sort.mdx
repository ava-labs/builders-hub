## Overview

Sort is a blockchain development platform that streamlines smart contract development workflows. With intuitive tools for contract development, testing, and deployment, Sort helps developers build and ship blockchain applications faster on Avalanche and other EVM-compatible networks.

## Features

- **Smart Contract IDE**: Browser-based development environment
- **Contract Templates**: Pre-built templates for common use cases
- **Testing Tools**: Comprehensive testing capabilities
- **Deployment Management**: Streamlined deployment workflows
- **Contract Verification**: Automated contract verification
- **Collaboration**: Team collaboration features
