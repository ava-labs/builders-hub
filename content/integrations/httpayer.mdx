## Overview

HTTPayer is a programmable payments layer that helps developers monetize APIs and enable autonomous, machine-to-machine payments using **HTTP 402 Payment Required** and the **x402 Protocol**. Built for Avalanche C-Chain, it supports fast settlement and predictable execution for payment-gated requests and agentic workflows.

## Features

- **HTTP 402 and x402-native payments**: Enable payment-gated API access using HTTP 402 Payment Required and the x402 Protocol.
- **Programmable payment rules**: Define pricing, limits, and policies for automated consumption by apps and agents.
