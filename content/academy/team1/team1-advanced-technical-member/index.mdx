## Welcome to the Advanced Technical Team1 Path

The **Technical Team1** course defined a baseline: every Team1 member should understand blockchain and Avalanche fundamentals, and Technical members should be comfortable with Solidity, ERC standards, frontend integration, and custom L1s. 
This **Advanced Technical Team1** course builds on that foundation with three additional pillars:

- **Interoperability on Avalanche (ICM & ICTT)** – sending messages and tokens between Avalanche L1s.[`https://build.avax.network/academy/avalanche-l1/interchain-messaging`](https://build.avax.network/academy/avalanche-l1/interchain-messaging) [`https://build.avax.network/academy/avalanche-l1/erc20-bridge`](https://build.avax.network/academy/avalanche-l1/erc20-bridge) [`https://build.avax.network/academy/avalanche-l1/native-token-bridge`](https://build.avax.network/academy/avalanche-l1/native-token-bridge)
- **Encrypted tokens (eERC)** – building privacy‑preserving applications with encrypted ERC standards.[`https://build.avax.network/academy/blockchain/encrypted-erc`](https://build.avax.network/academy/blockchain/encrypted-erc)
- **x402 payment infrastructure** – enabling HTTP‑native payments for web and AI agents on Avalanche.[`https://build.avax.network/academy/blockchain/x402-payment-infrastructure`](https://build.avax.network/academy/blockchain/x402-payment-infrastructure)

The goal is not to reproduce all of the protocol specifications inside this course, but to:

- Define the **knowledge expectations** for Advanced Technical Team1 members.
- Point you to the canonical Builder Hub courses and Team1 resources that cover each topic.
- Help you connect these technologies back to **Team1 events, workshops, and ecosystem support**.

Prerequisite: you should already be comfortable with the content in **“Become a Technical Team1 Member”**.
